Credit Loan Assessment API


1] Open a new window in current browser : .../main/view ids


2] Pick one of the IDs, let's say 387237


3] Go back to main page and enter a 6 digits customer ID in browser with URL as follows : ...main/predict?ID=123456


4] For our example, predict via URL will look like : .../main/predict?ID=387237


5] You should get as result : "Customer ID": 387237, "Default probability %": 6.0
